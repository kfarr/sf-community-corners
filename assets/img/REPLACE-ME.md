# Image placeholders — swap these for real photos

All `.svg` files here are **hand-drawn placeholders** so the site looks finished before you have
real photography. Replace them with real images (keep the same filename, or update the path in
`index.html` — every image slot is marked with a `<!-- TODO -->` comment).

| File | Used for | Suggested real photo | Target size |
|------|----------|----------------------|-------------|
| `hero-poster.svg` | Hero background / video poster | Foggy SF corner at golden hour, native plants in bloom | 1600×900 (16:9), or match your video |
| `example-planters.svg` | "Examples" card 1 | A real corner with wine-barrel / native-plant planters | 800×600 (4:3) |
| `example-mural.svg` | "Examples" card 2 | A street mural inside a painted safety zone | 800×600 |
| `example-cortland.svg` | "Examples" card 3 | The Cortland Ave (District 9) pilot corner | 800×600 |
| `example-hoboken.svg` | "Examples" card 4 | A daylit intersection with clear sightlines | 800×600 |

**Tips**
- Export real photos as optimized `.webp` or `.jpg` (~150–300 KB each). Update the `src` + `alt` in `index.html`.
- Keep `alt` text descriptive — it's read by screen readers and shown if an image fails to load.
- Hero image should be darker / lower-contrast in the upper-left where the headline sits, or the
  CSS overlay will keep text readable for you.
